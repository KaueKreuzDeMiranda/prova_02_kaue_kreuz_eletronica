# ⚡ Calculadora Elétrica

O projeto é uma ferramenta de terminal/navegador desenvolvida em JavaScript puro para facilitar cálculos fundamentais da eletricidade. O projeto abrange desde a famosa Lei de Ohm até o cálculo de consumo mensal e identificação de resistores por código de cores.

## 🚀 Funcionalidades

O projeto é dividido em módulos especializados, acessíveis através de um menu interativo:

* **Consumo Mensal**: Calcula o gasto de energia de aparelhos (em kWh) e o valor em Reais baseando-se no tempo de uso e potência (W ou kW).
* **Conversão de Unidades**: Converte grandezas entre múltiplos e submúltiplos do Sistema Internacional (de Nano a Giga).
* **Código de Cores de Resistores**: Identifica o valor de resistência e tolerância para resistores de 4 ou 5 faixas.
* **Lei de Ohm**: Resolve equações para Tensão ($V$), Corrente ($I$) ou Resistência ($R$).
* **Resistência Equivalente ($R_{eq}$)**: Calcula a resistência total de circuitos configurados em série ou em paralelo.

## 🛠️ Tecnologias Utilizadas

* **HTML**: Estrutura base para carregamento dos scripts.
* **JavaScript**: Lógica principal, manipulação de arrays, objetos e estruturas condicionais.
